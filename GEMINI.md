# Project: Niri Tonic OS

## Description

`niri-tonic-os` located here [https://github.com/tdesaules/chez-moi](https://github.com/tdesaules/chez-moi) is a `BlueBuild` [https://blue-build.org](https://blue-build.org) customized `Fedora Atomic`[https://fedoraproject.org/atomic-desktops](hhttps://fedoraproject.org/atomic-desktops/) based on a base `Universal Blue` [https://universal-blue.org/](https://universal-blue.org/) base image using `Niri` windows manager [https://niri-wm.github.io/](https://niri-wm.github.io/) and `Dank Linux` [https://danklinux.com/](https://danklinux.com/) with a `Nord` [https://www.nordtheme.com/](https://www.nordtheme.com/) theme applied.

## Usage

The Desktop is configured for those usage :

- Hardware : 
  - ProArt PX13 (HN7306)
  - AMD Ryzen AI MAX+ 395 Processor
  - AMD XDNA NPU
  - AMD Radeon Radeon 8060S Graphics 
  - 128GB LPDDR5X (96 VRAM + 32 RAM)
- Software : 
  - Local LLM with `Lemonade` [https://lemonade-server.ai](https://lemonade-server.ai)
  - Gaming using `Steam`
  - Developement, DevOps and Site Reliability
